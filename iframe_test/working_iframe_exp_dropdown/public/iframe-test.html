Test Application with Embedded Pulse

This page demonstrates how the PulseEmbedded component works within an iframe in another application.

Test Instructions:

  1. Search within iframe: Try typing a question in the search bar above
  2. Click predefined questions: Click on any of the blue suggestion buttons
  3. Test breakout: When you search, it should open the full result page in this parent window
  4. Test back navigation: Click the back button in the result page - it should return to this test page
  5. Verify functionality: All search features should work seamlessly

Expected behavior: When you perform a search in the iframe, it should break out and show the full result page with your query parameters. The back button should bring you back to this page.

  • Main Application - Full Pulse application
  • Embedded Version - Standalone embedded page
