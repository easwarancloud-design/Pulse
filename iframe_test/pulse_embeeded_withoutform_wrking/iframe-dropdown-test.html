Iframe Dropdown Visibility Test

Test Objective:

Verify that the dropdown suggestions are visible within iframe constraints when clicking on the search input.

Test Instructions:

  1. Click on the search input in the iframe below
  2. Check if the "Previous Conversations" dropdown appears
  3. Verify the dropdown is positioned correctly and visible within the iframe
  4. Try typing to trigger suggestions
  5. Click on predefined questions to test form submission

Small Iframe (400px height) - Most Constrained

Medium Iframe (600px height) - Moderate Constraints

Large Iframe (800px height) - Less Constrained

Expected Behavior:

  • Dropdown Positioning: In iframe context, dropdown should use fixed positioning from bottom of viewport
  • Size Adaptation: Reduced height (150px max) and padding for iframe context
  • Visibility: Dropdown should be fully visible regardless of iframe size
  • Form Submission: Clicking questions should break out of iframe using target="_top"
